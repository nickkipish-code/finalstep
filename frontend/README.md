# Frontend - Virtual Fitting Room

Next.js 14 frontend для виртуальной примерки одежды.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно на http://localhost:3000

## Сборка для продакшена

```bash
npm run build
npm start
```

## Структура

- `app/` - Next.js App Router страницы
- `components/` - React компоненты
- `hooks/` - Custom React hooks
- `public/` - Статические файлы

## Технологии

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Axios
